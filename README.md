# ⏳ 时薪桌面钟 (Hourly Wage Desktop Clock)

> 输入月薪、工作天数、工作时间段，实时看到每秒薪资累加——像沙漏一样，让你的时间价值肉眼可见。

## 🎯 产品定位

一款多平台个人效率工具，帮助打工人直观感知"每一秒值多少钱"。

## 🧩 核心功能

| 功能 | 说明 |
|------|------|
| 💰 薪资输入 | 月薪、月工作天数、日工作时间段 |
| ⏱ 实时计时 | 每秒计算并累加薪资，沙漏式增长 |
| 🕐 智能检测 | 自动判断工作时间状态（工作前/工作中/工作后） |
| 📱 多平台 | 桌面网页 / 桌面App / 移动网页 / 移动App |

## 🔄 三种时间状态

```
工作前 → 显示 0.00，暂停等待工作时间到达
工作中 → 每秒累加，实时显示已赚薪资
工作后 → 显示今日日薪总额，暂停计时
```

## 🛠 技术栈（规划中）

- 前端：HTML/CSS/JS → React/Next.js
- 桌面App：Electron / Tauri
- 移动App：PWA / React Native
